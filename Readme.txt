This script was written in Python3 and fully tested with the 
model Cisco IOS switches listed at the end of this document. 
This script is written in Python3 and has not been tested in 
Python2.  Please see Readme.txt files in Devices and Code Versions 
folders for correct formatting on input variables that are
declared in the files listed in the perspective folders.  
Log folder must also be present in parent directory for the script
to run.

Prereqs:
Python3
Netmiko
Models to be upgraded
Hostnames/IPs of target switches
MD5 Hash for all IOS versions to be used
TFTP server and it's IP

3550
3560G
